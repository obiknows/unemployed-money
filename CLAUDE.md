# Commit Convention

All commits must follow the [Conventional Commits](https://www.conventionalcommits.org/) standard:

```
<type>[optional scope]: <description>

[optional body]

[optional footer(s)]
```

**Types:** `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`, `perf`, `ci`, `build`

**Examples:**
- `feat: add blog post listing page`
- `fix(content): correct content.config loader path`
- `chore: migrate to astro v6 content config`
